# 📰 Fake News Detection using BERT

## 🔍 Problem Statement

Fake news detection is essential to prevent misinformation in digital media. This project aims to classify news as Fake or Real using NLP techniques.

---

## ⚙️ Approach

* Data preprocessing and cleaning
* Tokenization using HuggingFace Transformers
* Fine-tuning pretrained Transformer models

---

## 🤖 Models Used

* BERT (baseline)
* DistilBERT (optimized)

---

## 📊 Results

* Accuracy: **~94.5%**
* Evaluated using:

  * Precision
  * Recall
  * F1-score
  * Confusion Matrix

---

## 🚀 Deployment

* Built an interactive app using Gradio
* Users can input text and get predictions in real-time

---

## 🔧 Improvements

* Switched from BERT to DistilBERT for faster inference
* Reduced training time significantly

---

## 📚 Key Learnings

* Transformer-based NLP models
* Debugging real-world ML issues
* Model deployment
